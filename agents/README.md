# Datamak Lite Agent Notes

This folder contains instructions for agents working on Datamak Lite.

Start with:

- `workflow.md` for the development process, architecture boundaries, and first
  implementation loop.
- `maintenance_protocol.md` for the agent/manual operating contract that keeps
  Datamak Lite updated during daily campaign work.
- `../docs/new_code_onboarding.md` for a code-agnostic guide to adding a new
  simulation code or workflow.
- `../design/design_plan.md` for the product direction and first generic test
  graph.

These files are meant to guide implementation.  They are not runtime
configuration.
