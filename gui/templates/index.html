Train Surrogate Model

Algorithm. A RandomForest regressor (ensemble of decision trees) is trained on the selected gk_input features and predicts the chosen gk_run target. Uncertainty is estimated from the tree-to-tree spread (standard deviation).
Use ALL to include every status.
  • Test size: Fraction of rows held out for validation. 0 trains on all data and skips validation metrics.
  • Trees: Number of trees in the forest. More trees usually improve stability but take longer.
  • Max depth: Maximum depth of each tree. Blank/None allows fully grown trees.
  • Min samples leaf: Minimum rows required in a leaf node. Higher values smooth the model.
  • log1p(target): Train on log(1 + target) to reduce skew. Predictions are back-transformed; requires non-negative targets.
